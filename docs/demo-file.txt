## Fitur yang bisa dicoba:

2. 📤 Upload file ini ke folder yang dibuat

4. 🗑️ Hapus file dengan password
5. 📤 Export data untuk backup
6. 📥 Import data untuk restore

## Tips:

- File ini berukuran kecil, cocok untuk testing
- Coba upload berbagai jenis file
